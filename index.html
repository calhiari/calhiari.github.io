PIC Microcontrollers.

A really basic overview.

PIC Microcontrollers are programmable devices that can control electronic circuits, with imput and output signals.

They are often confused as microprocessors, and in fact they have a built-in microprocessor, but they also have

  • RAM memory
  • EEPROM memory
  • registers for A/D conversion
  • registers for PWM signal
  • registers for serial communication

and a lot more useful features as a control device.

It processing power is very below our usual personal computers, with just a few millions instructions per second, but if you put the right code on it, you can control everything you want to.

You can find every detailed information you want on Microchip's Website.
